# Changelog

## 0.1.0 - 2026-06-14

### Features

- Publish the initial `png-to-frontend-rebuild` skill package.
- Add mandatory asset planning for PNG/JPG/WEBP UI rebuilds.
- Require Image-2 or an available image model for custom visual assets when requested.
- Add rules that prevent screenshot icon cropping from being presented as redrawing.
- Add mobile screenshot guidance to avoid recreating OS status bars by default.
- Add ruler comparison workflow and calibration guidance.
- Include a reusable HTML template and framework adaptation references.

### Documentation

- Add user-facing README with installation, usage, expected outputs, asset rules, and versioning guidance.
